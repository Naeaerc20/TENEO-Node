# TENEO NODE

Welcome to Teneo Auto Node Script - This code gonna help you handle various instances(devices) connection for each account + multiple account operation

## Requieriments

``` shell

1. Buy proxies from 922 proxy provider here  https://center.922proxy.com/Dashboard/LongActingTraffic
2. Set their settings in this way - https://prnt.sc/C-5vLN03ix-c
3. Convert them to socks5 format with respectively prompt
4. Insert your 2CAPTCHA API on "captcha.py" using "nano scripts/captcha.py"

```

## Project Structure

``` shell

TENEO-Node/
├── index.js             // Main file that orchestrates the project flow.
├── node_modules/        // Directory containing all installed dependencies.
├── package.json         // Project configuration, scripts, and dependency list.
├── package-lock.json    // Dependency lock file ensuring reproducible installations.
├── README.md            // General documentation and project instructions.
├── scripts/             // Directory for auxiliary scripts.
│   ├── apis.js         // Contains API logic, authentication, and WebSocket connection.
│   └── captcha.py      // Python script that solves captchas using TwoCaptcha.
└── utils/               // Directory for utilities and configuration files.
    ├── bearers.json    // Stores access tokens (bearers) obtained from the login process.
    ├── converter.js    // Converts proxies from default_proxies.txt to proxies.txt.
    ├── default_proxies.txt  // Default list of proxies.
    ├── proxies.txt     // Final list of proxies to be used (generated by converter.js).
    ├── userdata.json   // Contains user account information.
    └── add_data.js     // Script that adds new accounts to userdata.json using Inquirer.

```

## Instructions

``` shell

1. git clone https://github.com/Naeaerc20/TENEO-Node
2. cd TENEO-Node
3. npm install
4. run any of the following prompts to interact with the application

- npm start - runs main script "index.js"
- npm run aggregate - runs "add_data.js" a file dedicated to help you add your existing accounts in the code
- npm run proxy - this script open "default_proxies.txt" where should paste your proxies obtained from 922 service
- npm run converter - this command converts all proxies added in "default_proxies.txt" in socks5 format and save them on "proxies.txt" allowing the code to work well

```
